# Defend Motos - Web catálogo

Proyecto web estático para trabajar en Codex, VS Code, Cursor o cualquier editor.

## Estructura

- `index.html`: página principal.
- `assets/logo-defend-motos.png`: logo de la marca.
- `data/products.json`: base editable de productos.
- `data/settings.json`: datos de contacto, horarios y redes.

## Cómo verlo localmente

Abrí `index.html` en el navegador.

También podés usar una extensión como Live Server en VS Code.

## Cómo publicarlo

Podés subir toda esta carpeta a Netlify, Vercel o GitHub Pages.

En Netlify:
1. Entrar a Netlify.
2. Add new site.
3. Deploy manually.
4. Arrastrar la carpeta completa o el `.zip`.
5. Publicar.
